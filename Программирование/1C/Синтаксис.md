## Два знака равно в переменных
<mark style="background: #FFF3A3A6;">Пример:</mark> 
Переменная = Переменная1 = Переменная 2;

Читается как: 
Переменная = Истина, если Переменная1 = Переменная 2
и Переменная = Ложь, если Переменная2 не равна Переменная1

Тут первый знак "=" является присваиванием
второй знак "=" является сравнением

## Понятие класса

## Понятие объекта
Объект это экземпляр класса

## Понятие массива
Массив - это упорядоченный набор элементов. Они хранятся строго по порядку.
## Объекты в 1С
В 1С объекты делятся на три вида:
1. Объекты мета-данных (это данные, которые сохраняются в информационной базе). Они уже встроены, создавать их новые мы не можем.
![[Pasted image 20230404232212.png]]

2. Общие объекты.
Не хранятся в информационной базе.
Существуют только в тот момент, когда мы их вызываем в рамках процедуры или функции.
![[Pasted image 20230404232239.png]]

3. Универсальные коллекции. Это набор переменных, которые могут быть в виде различных типов данных. Обычно используются для переноса данных из одной функции в другую, либо для сбора множества данных в одну переменную. 
Не хранятся в информационной базе.
Существуют только в тот момент, когда мы их вызываем в рамках процедуры или функции.
![[Pasted image 20230404232528.png]]

Обращение к свойствам:
object.name

Обращение к действиям:
object.name()

## Структура (Словарь)
// Структура чаще всего используется для переноса параметров из одной функции в другую. <mark style="background: #BBFABBA6;">В большинстве случаев используется структура.</mark> 
// В 1С словарь называется структура. Отличие структуры от массива в том, что список значение не является структуризированным т.е. мы не можем обратиться к значению по индексу.
Аргументом в структуре должен быть "ключ". "Ключ" - это всегда строка.

Структура = Новый Структура;

Структура.Вставить("Страна", "Россия");
Структура.Вставить("Город", "Ижевск");

// Обращение к значению в структуре
Сообщить(Структура.Страна);

// Обращение к значению по ключу:
Сообщить(Структура\["Город"\]);

// Перебор элементов в структуре по ключу:
Для Каждого ЭлементСтруктуры Из Структура Цикл
  Сообщить(ЭлементСтруктуры.Ключ);
КонецЦикла;

// Перебор элементов в структуре по значению:
Для Каждого ЭлементСтруктуры Из Структура Цикл
  Сообщить(ЭлементСтруктуры.Значние);
КонецЦикла;

// Проверка на наличие элементов по ключу (которого нет)
Если Структура.Свойство("Дом") Тогда
  Сообщить(Структура.Дом);
КонецЕсли;

## Соответствие
// Используется для передачи данных из одной функции в другую.
// Отличие соответствия от структуры, что в соответствии в качестве ключа может использоваться не только строка, а любой тип данных.

Соответствие = Новый Соответствие;

// Можно визуализировать. У нее есть строки и колонки, колонки должны иметь наименования. Никаких ограничений по количеству строк нет.
