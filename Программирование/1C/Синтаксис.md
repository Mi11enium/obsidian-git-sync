# Синтаксис условия ЕСЛИ

<mark style="background: #FFF3A3A6;">Пример 1:</mark> 
Если (условие) Тогда
  // код если условие = Истина; 
ИначеЕсли (условие2) Тогда
  // код если условие2 = Истина;
Иначе 
  // код если условие ложъ;
КонецЕсли;
