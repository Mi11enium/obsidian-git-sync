# Арифметические операторы

| Operator | Name           | Example |Return
| -------- | -------------- | ------- |
| +        | Addition       | x + y   | int
| -        | Subtraction    | x - y   | int
| *        | Multiplication | x * y   | int
| /        | Division       | x / y   | float
| %        | Modulus        | x % y   | int
| **       | Exponentiation | x ** y  | int/float
| //       | Floor division | x // y        | int

Examples:
~~~python
print(10 + 4) # Addiction
output: 14

print(10 - 4) # Subtraction
output: 6

print(10 * 4) # Multiplication
output: 40

print(10 / 4) # Division
output: 40

print(10 % 4) # Division
output: 40
