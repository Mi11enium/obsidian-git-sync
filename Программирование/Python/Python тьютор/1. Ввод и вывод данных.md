# Арифметические операторы

| Operator | Name           | Example |
| -------- | -------------- | ------- |
| +        | Addition       | x + y   |
| -        | Subtraction    | x - y   |
| *        | Multiplication | x * y   |
| /        | Division       | x / y   |
| %        | Modulus        | x % y   |
| **       | Exponentiation | x ** y  |
| //       | Floor division | x // y        |

Examples:
~~~python
print(10 + 4) # Addiction
output: 14

print(10 - 4) # Subtraction
output: 6

print(10 * 4) # Multiplication
output: 40

print(10 / 4) # Division
output: 40
